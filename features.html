KEY FEATURES

핵심 기능

🔄
MULTI-CHANNEL

멀티마켓 상품 일괄 등록

엑셀 한 장으로 국내외 수십 개 플랫폼에 상품을 일괄 등록하세요. AI가 카테고리를 자동 매핑하고, 키워드와 이미지 태그를 추천하여 등록 시간을 획기적으로 단축합니다.

  • 멀티마켓 상품 일괄 등록 (Amazon, Shopee, Lazada, TikTok Shop, 쿠팡, 네이버 등)
  • 플랫폼별 카테고리 자동 매핑 (AI 기반)
  • AI 키워드 추천 & 이미지 태그 자동 생성
  • 상품 정보 일괄 수정 & 가격 동기화
  • 이미지 자동 리사이징 & 워터마크
📋
ORDER MANAGEMENT

주문 관리

멀티마켓 주문을 자동으로 수집하고 주문 상태를 통합 조회합니다. 일괄 처리로 운영 효율을 극대화하세요.

  • 멀티마켓 주문 자동 수집 (국내외 전 채널)
  • 주문 상태 통합 조회 대시보드
  • 주문 일괄 처리 (확인, 발송, 취소)
  • 주문 이력 및 통계 분석
📦
INVENTORY

재고 관리

멀티채널 재고를 실시간으로 연동하여 품절 사고를 방지합니다. AI가 판매 추세를 분석하여 적정 재고 수준을 예측합니다.

  • 멀티채널 재고 실시간 연동 및 동기화
  • 재고 부족 알림 (임계값 설정)
  • AI 재고 예측 (판매 추세 기반)
  • 창고별 재고 현황 대시보드
🚚
INVOICE & SHIPPING

송장 관리

송장 번호를 자동으로 입력하고 물류사와 실시간 연동합니다. 배송 추적 정보를 고객에게 자동으로 전달합니다.

  • 송장 번호 자동 입력 (마켓별 자동 전송)
  • 물류사 연동 (CJ, 한진, 우체국, 해외 물류사)
  • 배송 추적 실시간 모니터링
  • 배송 완료 자동 알림
📝
PURCHASE ORDER

발주 관리

재고 기준에 따라 발주를 자동으로 생성하고 공급사를 체계적으로 관리합니다. 발주 이력과 입고 현황을 한눈에 파악하세요.

  • 자동 발주 생성 (재고 임계값 기반)
  • 공급사 관리 및 단가 비교
  • 발주 이력 및 입고 현황 추적
  • 발주서 자동 생성 및 전송
💬
CS MANAGEMENT

CS 관리

마켓별 고객 문의를 한 곳에서 통합 관리합니다. 자동 응답 템플릿과 다국어 지원으로 글로벌 CS를 효율적으로 운영하세요.

  • 마켓별 문의 통합 수집 및 관리
  • 자동 응답 템플릿 (FAQ 기반)
  • 다국어 지원 (AI 번역 연동)
  • CS 처리 현황 및 응답률 통계
🔗
ERP INTEGRATION

ERP 연동

기존 ERP 시스템과 API로 연동하여 회계 및 세무 데이터를 자동으로 전송합니다. 이중 입력 없이 데이터를 통합 관리하세요.

  • 기존 ERP 시스템 API 연동 (이카운트, 더존 등)
  • 회계 / 세무 데이터 자동 전송
  • 매출 / 매입 데이터 실시간 동기화
  • 정산 데이터 자동 정리
🏭
LOGISTICS AUTOMATION

물류 자동화

출고 처리를 자동화하고 물류사를 최적으로 배정합니다. 반품 및 교환 처리까지 시스템이 자동으로 관리합니다.

  • 출고 자동 처리 (주문 확인 즉시 출고 지시)
  • 물류사 자동 배정 (비용/지역 최적화)
  • 반품 / 교환 자동화 처리
  • 국내외 풀필먼트 통합 관리
📊
BI ANALYTICS

BI 분석

매출, 광고 ROAS, CRM 데이터를 통합 분석합니다. 브랜드 성장 지표를 한눈에 파악하고 데이터 기반 의사결정을 내리세요.

  • 매출 분석 (채널별, 상품별, 기간별)
  • 광고 ROAS / CPM / CPC 분석
  • CRM 고객 분석 (구매 패턴, LTV)
  • 브랜드 성장 지표 대시보드
🤖
AI AUTOMATION

AI 기반 자동화

AI가 상품명, 키워드, 카테고리를 자동으로 추천하고, 다국어 번역과 최적 가격까지 제안합니다. 셀러의 반복 업무를 AI가 대신합니다.

  • AI 상품명 / 키워드 추천 (검색 최적화)
  • AI 카테고리 자동 매핑
  • AI 다국어 번역 (영어, 태국어, 베트남어 등)
  • AI 가격 제안 (경쟁사 분석 기반)
📋
EXPORT DECLARATION

수출신고 자동화

전자상거래 수출 구매확인서를 자동으로 발급하고, uTradeHub와 연동하여 수출 신고 전 과정을 자동화합니다. HS코드 자동 분류로 통관을 간소화합니다.

  • 전자상거래 수출 구매확인서 자동 발급
  • uTradeHub 연동 (수출 신고 자동 제출)
  • HS코드 자동 분류
  • 수출 실적 자동 집계 및 리포트
💰
GLOBAL MARGIN TOOL

글로벌 마진툴

환율, 수수료, 광고비, 물류비, 통관비용을 모두 반영한 정확한 마진 계산으로 수익성을 사전에 확인하세요. 플랫폼별 순이익을 한눈에 비교할 수 있습니다.

  • 환율 / 수수료 / 광고비 / 물류비 / 통관비용 반영
  • 판매가 시뮬레이션으로 수익성 사전 확인
  • 플랫폼별 순이익 비교 분석
  • 실시간 환율 자동 업데이트
DASHBOARD

📈

실시간 매출 현황

🛒

플랫폼별 주문 현황

쇼피, 틱톡샵 등 채널별 주문 상태 한눈에 파악

📦

통합 재고 관리

국내외 창고 재고를 실시간 동기화하여 품절 방지

💰

수익성 분석

플랫폼별 마진, 수수료, 물류비 상세 분석 리포트
