# 🚀 DIVOOST 홈페이지 GitHub Pages 배포 가이드

## 📦 파일 구성

```
divoost-website/
├── index.html              # 메인 페이지
├── css/style.css           # 스타일 (27KB)
├── js/
│   ├── i18n.js             # 다국어 처리
│   └── main.js             # 메인 로직
├── lang/
│   ├── ko.json             # 한국어
│   ├── vi.json             # 베트남어
│   └── en.json             # 영어
├── images/logo.jpg         # 로고
├── .nojekyll               # GitHub 빌드 설정
└── README.md               # 영문 배포 가이드
```

---

## ✅ Step 1: GitHub 계정 준비

1. https://github.com 접속
2. 계정이 없다면 "Sign up" 클릭하여 가입
3. 이메일 인증 완료

---

## ✅ Step 2: 저장소(Repository) 생성

1. GitHub 로그인 후 오른쪽 상단 **"+"** 아이콘 → **"New repository"** 클릭
2. 다음 정보 입력:
   - **Repository name**: `divoost-website` (또는 원하는 이름)
   - **Description**: `DIVOOST 공식 홈페이지` (선택사항)
   - **Public** 선택 (GitHub Pages 무료 사용을 위해 반드시 Public)
   - ⚠️ "Add a README file" 체크 **해제** (이미 파일에 포함되어 있음)
3. **"Create repository"** 버튼 클릭

---

## ✅ Step 3: 파일 업로드

### 방법 A: 웹에서 드래그 앤 드롭 (가장 쉬움) ⭐ 추천

1. 생성된 저장소 페이지에서 **"uploading an existing file"** 링크 클릭
2. `divoost-website.zip` 압축 해제 후 폴더 **내부의 모든 파일과 폴더**를 드래그해서 업로드
   - ⚠️ `divoost-website` 폴더 자체가 아닌, 그 **안의 내용물**을 업로드해야 함
   - 올바른 업로드: `index.html`, `css/`, `js/`, `lang/`, `images/`, `README.md`, `.nojekyll`
3. 하단 "Commit changes":
   - Commit message: `Initial commit`
4. **"Commit changes"** 버튼 클릭

### 방법 B: Git 명령어 사용 (개발자)

```bash
cd divoost-website
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/사용자명/divoost-website.git
git push -u origin main
```

---

## ✅ Step 4: GitHub Pages 활성화

1. 저장소 페이지 상단의 **"Settings"** 탭 클릭
2. 왼쪽 사이드바에서 **"Pages"** 클릭
3. **"Build and deployment"** 섹션에서:
   - **Source**: `Deploy from a branch` 선택
   - **Branch**: `main` 선택
   - **Folder**: `/ (root)` 선택
4. **"Save"** 버튼 클릭
5. 1~2분 대기 후 상단에 초록색 체크 표시와 함께 URL이 표시됨:
   ```
   ✅ Your site is live at https://사용자명.github.io/divoost-website/
   ```

---

## ✅ Step 5: divoost.com 도메인 연결 (선택사항)

GitHub Pages URL 대신 `divoost.com`으로 접속하게 하려면:

### 5-1. GitHub에서 설정

1. Settings → Pages에서 **Custom domain** 란에 `divoost.com` 입력
2. **"Save"** 클릭

### 5-2. 도메인 등록업체에서 DNS 설정

현재 `divoost.com`이 등록된 업체(예: Cafe24, Gabia, GoDaddy 등)에서 DNS 설정:

**루트 도메인 (divoost.com)용 A 레코드 4개 추가:**
```
Type: A    |  Name: @    |  Value: 185.199.108.153
Type: A    |  Name: @    |  Value: 185.199.109.153
Type: A    |  Name: @    |  Value: 185.199.110.153
Type: A    |  Name: @    |  Value: 185.199.111.153
```

**www 서브도메인용 CNAME 레코드 1개:**
```
Type: CNAME  |  Name: www  |  Value: 사용자명.github.io
```

### 5-3. 최종 확인

- 10분~1시간 대기 (DNS 전파 시간)
- GitHub Settings → Pages에서 **"Enforce HTTPS"** 체크
- `https://divoost.com` 접속 테스트

## 🔧 수정 방법

### 연락처 정보 변경

**전화번호를 나중에 추가할 때:**

1. `index.html` 파일 열기
2. `<!-- Contact Section -->` 부분에서 전화번호 항목 추가:
```html
<div class="contact-item">
    <div class="contact-label" data-i18n="contact.phoneLabel">PHONE</div>
    <a href="tel:+84000000000" class="contact-value">+84 000 000 000</a>
</div>
```
3. `lang/ko.json`, `lang/vi.json`, `lang/en.json`의 `contact` 섹션에 `phoneLabel` 추가

### 온라인 채널 링크 추가 (Shopee, Facebook, YouTube 등)

`index.html`의 `<footer>` 부분에 SNS 링크 섹션 추가

### 로고 교체

`images/logo.jpg`를 새 파일로 교체 (같은 이름 유지)

### 텍스트 수정

`lang/ko.json`, `lang/vi.json`, `lang/en.json` 파일에서 해당 언어의 텍스트만 수정하면 됨
- 예: 메인 제목 수정 → `hero.title1`, `hero.title2` 값 변경

### 색상 변경

`css/style.css` 파일 최상단 `:root` 섹션에서:
```css
--color-gold: #f5b82f;      /* 메인 포인트 색상 */
--color-black: #0a0a0a;     /* 배경 */
--color-blue: #0d1b2a;      /* 보조 배경 */
```

## ❓ 자주 묻는 질문

### Q: GitHub Pages 무료인가요?
A: 네, Public 저장소는 완전 무료입니다. 트래픽 소프트 제한 100GB/월입니다.

### Q: 이메일 문의 폼은 어떻게 작동하나요?
A: 현재 `mailto:` 링크로 사용자의 기본 이메일 앱이 열립니다. 실제 서버로 전송하려면:
- **Formspree** (무료): https://formspree.io
- **EmailJS** (무료): https://www.emailjs.com

### Q: 언어 자동 감지를 할 수 있나요?
A: 네. `js/i18n.js`에서 `navigator.language`로 기본 언어 자동 감지 추가 가능합니다. 원하시면 다음 업데이트에서 추가하겠습니다.

### Q: 수정 후 바로 반영되나요?
A: GitHub에 파일 업로드 후 약 1~2분 후에 반영됩니다 (브라우저 강제새로고침: Ctrl+F5 / Cmd+Shift+R).

### Q: 검색엔진(구글/네이버)에 등록하려면?
A: 배포 후 Google Search Console, 네이버 서치어드바이저에 사이트 등록하면 됩니다.
