## 3. 기능 (Features) — AI 자동화 기능

### 3-1. AI 상품 분석 (출처 image3)
- 헤드라인: 이 계절에는 어떤 상품이 팔릴까? 빅데이터 기반 AI가 팔릴 만한 상품을 제안해 드립니다.
- 마켓별 분석 카드 (예시값):
  - **스마트스토어**: 상품수 5,947,308 / 해외상품수 2,394,222 / 총 리뷰수 24,282 / 최대 리뷰수 8,278
  - **쿠팡**: 상품수 499,831 / 평균가 33,004원 / 총 리뷰수 52,550 / 로켓대상 22.22% / 제트배송 58.33%
  - 보조지표: 모바일 94%·PC 6%, 여성 95%·남성 5%, 월 검색량 35,360, 경쟁강도 17.34, 상품 평균가 55,988원, 계절성 등
- 메시지: 마켓별 상품등록 분석·평균가·리뷰수 등 디테일 정보를 제공해 더 신중한 결정을 돕습니다.

### 3-2. AI 상품 추천 (출처 image4)
- 헤드라인: 빅데이터 기반 **HubOnTrade AI**가 판매 가능성이 높은 상품을 찾아 분석하고 추천해 드립니다.
- 서브: 타오바오·티몰·1688에 직접 가지 않아도 한번에 추천 키워드 상품을 자동으로 찾아드립니다.
- 탭: 타오바오 / 티몰 / 1688

### 3-3. 엑셀 상품수집 (출처 image5·6·7)
- 라벨: 엑셀 상품수집
- 헤드라인: **유연한 상품 소싱, 엑셀 수집으로 더욱 편리하게!**
- 서브: 소싱한 상품을 엑셀로 관리하시나요? 복잡한 설정 없이 업로드만 하면 수집이 완료됩니다.
- 사용 흐름: `HubOnTrade 엑셀 양식.xlsx` 다운로드 → 엑셀 파일 업로드 → 엑셀 소싱 실행
- 직접 입력 컬럼: 상품 URL / 상품명 / 키워드 / 카테고리
- 보조: 수집 중 다른 작업도 가능, **언제든지 수집현황 확인**(수집완료 표시)

### 3-4. 쉽고 편리한 상품관리 (출처 image8·9·10)
- 헤드라인: 상품 수집부터 업로드까지, **가장 쉽고 빠른 방법**
- 서브: 클릭 한 번으로 AI가 최적의 **카테고리·상품명·키워드**를 자동 생성. 손쉬운 상품관리를 HubOnTrade와 함께.
- 손쉬운 프로그램: 초급~중급 판매자 모두에게 적합 (예: "타프" 검색 → 스포츠/레저 > 캠핑 > 타프/그늘막 자동 분류)
- 자동 설정: AI 상품명 · AI 키워드 · AI 검색속성(색상계열·사용계절·설치방식 등)
- **옵션명 25자 자동보정**: 너무 긴 옵션명을 AI가 자동으로 25자 이내로 깔끔하게 정리

## 4. 서비스 (Service) — 주문·배대지·운영

### 4-1. 원클릭 관리 / 배대지 (출처 image21·22·23)
- 라벨: 원클릭 관리
- 헤드라인: **소문난 HubOnTrade 배대지! 원클릭 주문 연동의 편리함**
- 서브: 배대지 주문, 클릭 한 번으로 단 1초 만에 완료! 실시간 알림·관리로 언제 어디서나 빠르게 처리.
- **한눈에 파악되는 주문현황**(예시 라벨): 신규주문배달 · 준비중 · 품절이슈관리 · 구매중 · 구매후발송대기 · 현지배송중 · 패킹완료 · 통관중 · 국내배송중 · 배송완료 · 배송취소
- **구매시점부터 전 과정 배송추적**: Shipping Tracking (예: CJ대한통운 운송장 실시간 조회)
- **안심케어 검수**: 배송 중 파손도 HubOnTrade 배대지의 무료 디테일 검수·사진 촬영으로 불량을 찾아드립니다.

### 4-2. 구매대행 운영 서비스 8종 (출처 image24)
1. **자동 주문수집** — 마켓 주문 자동 확인 + 실시간 알림
2. **마켓 운송장 전송** — 클릭 한 번으로 운송장 자동 전송 → 배송중 전환
3. **카카오 알림톡** — 배송 상황을 구매 고객에게 카카오로 자동 안내
4. **구매상품 결제대행** — 1688 등 도매처 입금이 어려운 구매 결제를 대행
5. **현지 반품처리** — 불량·오배송 걸러지면 현지 판매점으로 반품 진행
6. **대량 구매지원** — 대량구매를 HubOnTrade에서 간편하게 관리
7. **자동 합배송** — 동일 수취인이면 자동으로 합배송 연결
8. **자동 소싱자료 파일** — 매입·매출을 사용만으로 자동 정리(엑셀)
