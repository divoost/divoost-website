⚙ Resend.com API (선택)

Resend.com에서 API 키를 발급받아 등록하면 클라이언트에서 직접 이메일 발송이 가능합니다.
무료 플랜: 월 3,000건, 100건/일

📘 Supabase Custom SMTP 설정 가이드

1. Supabase Dashboard → Authentication → Email Templates → SMTP Settings
2. "Enable Custom SMTP" 토글 ON
3. 다음 정보 입력:
　- Host: smtp.gmail.com (Gmail) 또는 smtp.sendgrid.net 등
　- Port: 587
　- Username: 본인 이메일
　- Password: Gmail 앱 비밀번호 (일반 비밀번호 아님)
4. Gmail 앱 비밀번호 발급: myaccount.google.com/apppasswords
