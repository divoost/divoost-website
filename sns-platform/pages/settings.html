🔧 일반 설정

📡 SNS 채널 API 설정

각 SNS 플랫폼의 API 키를 등록하면 자동 포스팅, 분석, 댓글 관리 기능을 사용할 수 있습니다.

📘
Facebook
페이지/그룹 자동 포스팅
연결 안됨
📌 Facebook Developers에서 앱을 생성하고, Graph API Explorer에서 Page Access Token을 발급받으세요.
🎵
TikTok
비즈니스 계정 콘텐츠 관리
연결 안됨
📌 TikTok for Developers에서 앱을 등록하고 Content Posting API 권한을 신청하세요.
📸
Instagram
비즈니스/크리에이터 계정
연결 안됨
📌 Instagram API는 Facebook Graph API를 통해 접근합니다. Facebook 앱에서 Instagram Basic Display 또는 Instagram Graph API를 활성화하세요.
📗
네이버 블로그
블로그 자동 글쓰기
연결 안됨
📌 네이버 개발자센터에서 애플리케이션을 등록하고 블로그 API 사용 신청을 하세요. OAuth 2.0 인증이 필요합니다.
☕
네이버 카페
카페 자동 글쓰기
연결 안됨
📌 네이버 카페 API는 블로그와 동일한 Client ID/Secret을 사용하되, 카페 글쓰기 권한이 추가로 필요합니다.
📕
샤오홍수 (小红书)
콘텐츠 발행 및 분석
연결 안됨
📌 小红书 Open Platform (open.xiaohongshu.com)에서 개발자 등록 후 앱을 생성하세요. 중국 사업자등록증이 필요할 수 있습니다.
🎬
도우인 (抖音)
숏폼 영상 콘텐츠 관리
연결 안됨
📌 抖音 Open Platform (open.douyin.com)에서 앱을 등록하세요. 중국 사업자 또는 개인 인증이 필요합니다.

🤖 AI 설정

📌 AI API 키를 등록하면 더 고품질의 콘텐츠를 자동 생성할 수 있습니다. 미등록 시 내장 템플릿 기반으로 생성됩니다.
